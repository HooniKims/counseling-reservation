# 변경 이력

## v2.0.0 - 2024-11-17

### 새로운 기능 ✨

#### 교사 기능
- **예약 취소**: 교사가 예약을 직접 취소할 수 있는 기능 추가
  - 예약 카드에 취소 버튼 추가
  - 취소 시 슬롯 상태 자동 복구 (reserved → available)

- **Excel 내보내기**: 예약 데이터를 Excel 파일로 다운로드
  - 학번, 이름, 날짜, 교시, 시간, 주제, 내용 포함
  - 파일명: `상담예약_YYYY-MM-DD.xlsx`

#### 학부모 기능
- **예약 확인 페이지** (`/check-reservation`): 새로운 페이지 추가
  - 학번과 이름으로 예약 조회
  - 예약 내역 목록 표시
  - 지난 예약과 예정된 예약 구분

- **예약 취소**: 학부모가 본인의 예약을 직접 취소 가능
  - 지난 예약은 취소 불가
  - 예정된 예약만 취소 버튼 활성화

#### UI/UX 개선
- **메인 랜딩 페이지** 개선
  - 교사용/학부모용 선택 카드 디자인
  - 각 기능별 안내 및 설명
  - 사용 가이드 추가

- **예약 완료 화면** 개선
  - 예약 확인 페이지 링크 추가

### 기술적 변경 🔧
- `xlsx` 패키지 추가 (Excel 내보내기)
- 예약 조회 쿼리 최적화 (학번 + 이름 복합 조회)

### 파일 변경
- `app/page.tsx`: 메인 랜딩 페이지로 변경
- `app/teacher/page.tsx`: 예약 취소 및 Excel 내보내기 기능 추가
- `app/booking/[teacherId]/page.tsx`: 예약 확인 페이지 링크 추가
- `app/check-reservation/page.tsx`: 새로운 페이지 생성

## v1.0.0 - 2024-11-17

### 초기 릴리스 🎉

#### 교사 기능
- 교시 시간 커스터마이징 (1-7교시)
- 달력 기반 날짜 선택
- 날짜별 교시 선택
- 상담 가능 시간 설정
- 실시간 예약 현황 확인
- 학부모 예약 링크 생성 및 복사

#### 학부모 기능
- 2단계 예약 프로세스
- 학생 정보 입력 (학번, 이름)
- 실시간 예약 가능 시간 표시
- 상담 주제 선택 (학업, 진로, 교우관계, 기타)
- 상담 내용 작성
- 예약 완료 확인

#### 기술 스택
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Firebase Firestore
- Lucide React (아이콘)

#### 주요 특징
- 실시간 데이터 동기화 (onSnapshot)
- 중복 예약 방지
- 반응형 디자인
- 접근성 고려
